Semantische Elemente

Heading

Das Element heading ist da um den Kopf einer Webseite zu bergen.

Main

Das Element main ist da um den Hauptinhalt einer Webseite zu bergen.

Footer

Das Element footer ist da um die Fußzeile einer Webseite zu bergen.

Section

Das Element section ist da um einen Teil ("eine Sektion") des Inhalts einer Webseite zu bergen.

article

Das Element article ist da um Artikel zu bergen.

Aside

Das Element aside ist da um Inhalte auf der rechten oder linken Seite einer Webseite darzustellen.

Details

Das Element details ist da um Zusatzinformationen zu einem Text per Klick darzustellen. Der Haupttext, welcher immer angezeigt wird, wird mit dem Element summary angegeben. Das und p kommen in details.

<details>
   <summary>Copyright 1999-2018</summary>
   <p>Copyright 1999-2018 by Vladbraa Data. All Rights Reserved.</p>
</details>

Figure

Das Element figure ist da um Textinformation zu einem Bild darzustellen. Die Textinformationen werden mit figcaption angegeben. Das und der img-tag kommen in figure.

<figure>
   <img src="historischs_haus.png" alt="Historisches Haus">
   <figcaption>Historisches Haus - Rom, Italien.</figcaption>
</figure>

Overflow

Oft werden Bilder in einem div außerhalb seiner Grenzen dargestellt, obwohl sich der img-tag innerhalb des divs befindet. Um dies zu vermeiden muss das div zu overflow: auto; gesetzt werden. Das nennt man Clearfix.

Flexbox

Mit display: flex; stellt man die Tochterelemente auf Flexbox ein.

Beispiel

body, html {
   box-sizing: border-box;
   margin: 0;
   font-family: 'Montserrat', sans-serif;
}
h1, h2, p {
   margin: 0;
}
nav, footer {
   color: #FFF;
   background-color: #333;
   padding: 30px;
}
nav:hover, footer:hover {
   background-color: #222;
}
main {
   display: flex;
}
#left, #right {
   color: #FFF;
   background-color: #666;
   padding: 30px;
   flex: 1;
}
#left:hover, #right:hover {
   background-color: #555;
}
#content {
   padding: 30px;
   flex: 3;
   display: flex;
}
.content-pieces {
   margin-right: 30px;
   min-height: 300px;
   background-color: #DDD;
   padding: 30px;
   flex: 1;
}
.content-pieces:hover {
   background-color: #CCC;
}
#last {
   margin-right: 0px !important;
}
