Datentypen

Short

Mit dem Datentypen short speichert man Zahlen mit einer maximalen Länger von 16 Bit, also von ungefähr -30.000 bis ungefähr 30.000. Man kann allerdings nur ganze Zahlen speichern.

short a = new short();
a = 30000;

Int

Mit dem Datentypen int speichert man Zahlen mit einer maximalen Länger von 32 Bit, also von ungefähr -2.000.000.000 bis ungefähr 2.000.000.000. Man kann allerdings nur ganze Zahlen speichern.

int b = new int();
b = 300000000;

Long

Mit dem Datentypen long speichert man Zahlen mit einer maximalen Länger von 64 Bit, also von ungefähr -10.000.000.000.000.000.000 bis ungefähr 10.000.000.000.000.000.000. Man kann allerdings nur ganze Zahlen speichern.

long c = new long();
c = 3000000000000000000;

Float

Der Datentyp float speichert Dezimalzalen mit einer Genauigkeit von bis zu 7. Man muss allerdings ein F an das Ende der Zahl stellen damit dich der Computer die beiden Datentypen float und double unterscheiden kann.

double h = new double();
h = 4.19472017593756;

Double

Der Datentyp double speichert Dezimalzalen mit einer Genauigkeit von bis zu 15.

float i = new float();
i = 4.394234F;

Bool

Der Datentyp bool kann entweder richtig oder falsch speichern.

bool d = new bool();
d = true;

String

In den meisten Fällen wird der Datentyp string verwendet um Text zu speichern.

string e = System.String.Empty;
e = "Hello World!";

Var

Der Datentyp var kann anstelle von allen anderen Datentypen verwendet werden. Das funktioniert indem der Compiler den expliziten Typ der Variable basierend auf der Verwendung bestimmt. Die kann ein bisschen Zeit und Code sparen, allerdings verliert man dafür ein bisschen Übersicht. var f = 5 führt dazu, dass der Compiler automatisch erkennt, dass 5 der Datentyp int ist und wird den Code im Wesentlichen für das Programm zu int f = 5 umschreiben.

var f = 3;

var f = "Hello World!";

var f = new Connection();

Char

Der Datentyp char wird meistens verwendet um einzelne Variablen zu speichern.

char g = new char();
g = 'X';

Arrays

Arryas sind Sammlungen bzw. Listen von Werten.
Alle gespeicherten Werte in einem Array müssen dem gleichen Datentyp entsprechen, welcher bereits bei der Initialisierung angegeben wird.
Arrays können eindimensional (Listen) oder multidimensional (Felder, 3D-Umgebungen, ...) sein.

(1D-)Arrays

Eindimensionale Arrays werden für klassiche Sammlungen von Werten verwendet. Man benötigt sie um z. B. mehrere ausgerechnete Werte zu speichern die zusammen gehören.

int[] j = new int[3];
j[0] = 3;
j[1] = 8;
j[2] = 4;

// 3
// 8
// 4

2D-Arrays

Zweidimensionale Arrays werden für "Felder von Werten" verwendet. Man benötigt sie um z. B. ein "Vier Gewinnt" Spiel in der Konsole zu simulieren.

int[,] k = new int[2, 2];
k[0, 0] = 6;
k[0, 1] = 2;
k[1, 0] = 1;
k[1, 1] = 9;

// 6 1
// 2 9

If/Else-Anweisung, While-Schleife und For-Schleife

If/Else-Anweisung

Eine If/Else-Anweisung führt eine Funktion aus wenn sie als wahr eingestuft wurde.

int x = 3;

if(x > 0)
{
   Console.WriteLine("X ist größer als 0.");
}
else
{
   Console.WriteLine("X ist kleiner als 0.");
}

X ist größer als 0.

if/else-Anweisungen können auch ohne Klammern geschrieben werden.

if(x > 0)
   Console.WriteLine("X ist größer als 0.");
else
   Console.WriteLine("X ist kleiner als 0.");

While-Schleife

Eine while Schleife führt so lange ihre Funktionen aus bis ihre Vorraussetzung nicht mehr wahr ist. Sie kann nicht ohne Klammern geschrieben werden.

int y = 3;

while (y > 0)
{
   Console.WriteLine ("Y ist größer als 0.");
   y--;
}

Y ist größer als 0.
Y ist größer als 0.
Y ist größer als 0.

For-Schleife

Die for Schleife verwendet man, wenn die Anzahl der Durchläufe bekannt ist. Es ist eine Zähler-gesteuerte Schleife. Sie hat 3 Bereiche.

Bereich 1: Startwert der Zählvariablen setzen, z. B. i = 0
Bereich 2: Durchlauf-Bedingung, z. B. i < 5
Bereich 3: Operation auf Zählvariable ausführen, z. B. i++

for (int i = 0, j = 1; i < 5; i++, j++)
{
   Console.WriteLine("Es ist Durchlauf " + j + "!");
}

Es ist Durchlauf 1!

Enums und Points

Enums

Ein enum besteht aus einer Reihe benannter ganzzahliger Konstanten. Ein Aufzählungstyp wird mit dem Schlüsselwort enum deklariert.
Um ein enum zu erstellen muss man in Visual Studio eine Klasse hinzufügen und class Example in public enum Example umändern.

using System;

namespace PlayField
{
   public enum FieldType
   {
      HEAVY_SWORDSMAN, ARCHER, LIGHT_INFRANTRY, SNIPER 
   }
}

int[,] m = new int[3, 3];
m[2, 2] = (int)FieldType.SNIPER;
Console.WriteLine(m[2, 2]);
Console.WriteLine((FieldType)m[2, 2]);

Points

Ein point ist eine der Verwendungszwecke für die Klasse struct. Er wird verwendet um zwei oder mehr geografische Koordinaten zu speichern.
Um einen point zu erstellen muss man in Visual Studio eine Klasse hinzufügen und class Example in public struct Point umändern.

using System;

namespace PlayField
{
   public struct Point
   {
      public int x; 
      public int y; 
   }
}

Point p = new Point();
p.x = 0;
p.y = 1;

int[,] field = new int[2, 2];
field[p.x, p.y] = 5;

// 0 0
// 5 0

{
   if (y >= 0 && y <= field.GetLength(1) && x >= 0 && x <= field.GetLength(0))

   }
